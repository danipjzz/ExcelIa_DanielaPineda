Este proyecto consiste en el análisis de ventas mensuales utilizando herramientas de análisis automático de Excel.
Se creó una tabla con los datos de ventas correspondientes a cada mes del año. A partir de esta información, se utilizaron herramientas como gráficos recomendados y tablas dinámicas para identificar patrones y tendencias en el comportamiento de las ventas.
La pregunta clave del proyecto fue:
¿En qué mes se registran mayores ventas y cómo se comporta la tendencia a lo largo del año?
A través del análisis visual se identificó una tendencia creciente, destacándose el último mes como el de mayor rendimiento.
Este proyecto demuestra cómo Excel, apoyado en herramientas inteligentes de análisis, facilita la interpretación de datos y la toma de decisiones.
